## Installation
#### The program is installed by accessing the GitHun Repo Professional-ReadMe-Generator at the Git Repo below downloaded and initialized in the command terminal of vs code through the index.js file and running the command  'node inex.js' this will begin the program. 

## Usage
#### The program is used by completing the prompts and saving the testREADME.md file that the program generates locally once you have completed the first run, each new run will overwrite the previos test README
